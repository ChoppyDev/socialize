Implementation example:

<div id="socialize-container" data-list="facebook,twitter,linkedin,email" data-href="www.socialize.io" data-light="yes"></div>
Html Attribute Args Description
data-list facebook,twitter,linkedin Generate the buttons passed as args (separated by comas)
data-href [url] | current Link for share button option
data-light (optionnal) yes | no Enable the light mode for buttons (disabled by default)
